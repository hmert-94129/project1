# FuelGuard Website

Modern teknolojiler ve Tailwind CSS kullanılarak geliştirilmiş FuelGuard kurumsal web sitesi.

## Teknolojiler

- **Next.js**: React tabanlı web framework
- **TypeScript**: Tip güvenliği için
- **Tailwind CSS**: Stil ve tasarım için
- **Framer Motion**: Animasyonlar için
- **React Icons**: İkonlar için

## Özellikler

- Modern ve responsive tasarım
- Performans odaklı geliştirme
- SEO dostu yapı
- Çoklu dil desteği
- Kolay yönetilebilir bileşen yapısı

## Kurulum

```bash
# Bağımlılıkları yükleyin
npm install

# Geliştirme sunucusunu başlatın
npm run dev

# Projeyi derleyin
npm run build

# Derlenmiş projeyi çalıştırın
npm run start
```

## Proje Yapısı

```
fuelguard-website/
├── public/            # Statik dosyalar
│   └── images/        # Görseller
├── src/               # Kaynak kodları
│   ├── app/           # Next.js App Router sayfaları
│   ├── components/    # React bileşenleri
│   ├── lib/           # Yardımcı fonksiyonlar ve kütüphaneler
│   └── styles/        # CSS dosyaları
├── .gitignore         # Git tarafından yok sayılacak dosyalar
├── next.config.js     # Next.js yapılandırması
├── package.json       # Proje bağımlılıkları
├── postcss.config.js  # PostCSS yapılandırması
├── tailwind.config.js # Tailwind CSS yapılandırması
└── tsconfig.json      # TypeScript yapılandırması
```
